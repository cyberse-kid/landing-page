Descripción General

SmartIRecruit es una plataforma full stack que automatiza y objetiviza el proceso de selección de personal combinando un filtro ATS inteligente con análisis de video en tiempo real. Durante las entrevistas grabadas, el sistema detecta y analiza emociones del candidato usando visión por computadora (DeepFace + OpenCV), reduciendo el sesgo humano en la evaluación. El backend integra Groq API como LLM para el análisis automatizado de CVs y respuestas, mientras el frontend en React ofrece un panel completo para reclutadores con métricas por candidato.
